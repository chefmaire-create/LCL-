# LCL- Simulation Banque

Une simulation simple et fictive de l'application bancaire **LCL** (Crédit Lyonnais).

Ce projet permet de tester des virements bancaires **sans aucun risque** : tout est faux, pas d'argent réel !

### Fonctionnalités
- Voir les soldes de comptes fictifs
- Ajouter des bénéficiaires
- Faire des virements (instantanés ou différés)
- Historique des opérations
- Interface inspirée de la vraie app LCL

### Pourquoi ce projet ?
Pour s'entraîner à faire des virements, comprendre comment ça marche, ou juste pour un projet web fun.

**Attention : Ceci n'est PAS l'application officielle LCL. Aucune connexion à la vraie banque !**

### Comment tester
1. Le site est déployé ici : (tu mettras le lien Vercel plus tard, ex: https://lcl-.vercel.app)
2. Ou en local :
   - `npm install`
   - `npm run dev`
   - Ouvre http://localhost:3000

Technologies : React / Next.js / Tailwind (ou ce que tu utilises vraiment)
